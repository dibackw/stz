3. Описание проекта (предметная область)
Название проекта: stz
Предметная область проекта — системы онлайн-тестирования знаний. Такие системы широко используются в образовательных учреждениях для проверки знаний студентов, проведения контрольных работ, экзаменов и самостоятельных тестов.
Традиционные формы тестирования (бумажные тесты) требуют большого количества времени на проверку и обработку результатов. Кроме того, хранение и анализ результатов в бумажном виде затрудняет статистическую обработку данных.
Разрабатываемое веб-приложение предназначено для автоматизации процесса тестирования знаний. Система позволит преподавателям создавать тесты, назначать их студентам, автоматически проверять ответы и сохранять результаты в базе данных.
Основные пользователи системы:
Преподаватели: создают тесты; добавляют вопросы; назначают тесты студентам; просматривают результаты.
Студенты: проходят тестирование; получают результаты; могут просматривать историю своих тестов.
Администратор: управляет пользователями системы; управляет тестами и предметами.
Приложение будет работать через веб-интерфейс, что позволит использовать систему с любого устройства, имеющего доступ к интернету.

5. Макеты интерфейса
Интерфейс системы должен быть простым и удобным для пользователей.
Страница авторизации. Элементы страницы: поле логин, поле пароль, кнопка «Войти», сообщение об ошибке при неправильном вводе данных.
Главная страница (панель управления)
После входа пользователь попадает на главную страницу системы.
Элементы страницы: меню навигации, список доступных разделов, информация о системе.
Основные разделы: Тесты, Вопросы, Результаты, Пользователи.
Страница создания теста. Элементы страницы: название теста, описание теста, кнопка «Добавить вопрос», кнопка «Сохранить тест».
Страница прохождения теста. Элементы страницы: название теста, список вопросов, варианты ответов, таймер выполнения, кнопка «Завершить тест».
Страница результатов. Элементы страницы: список тестов, полученные баллы, дата прохождения теста, статистика результатов.

6. Описание API
Формат данных: JSON
Аутентификация: JWT
GET /tests Получение списка тестов.
Пример ответа
{
  "data": [
    {
      "id": 1,
      "title": "Основы программирования",
      "questions_count": 10
    }
  ]
}
POST /tests Создание нового теста.
Пример запроса:
{
  "title": "Базы данных",
  "description": "Тест по основам SQL"
}
GET /questions Получение списка вопросов.
Пример ответа:
{
  "data": [
    {
      "id": 1,
      "question": "Что такое SQL?"
    }
  ]
}
POST /results Сохранение результатов тестирования.
Пример запроса:
{
  "user_id": 1,
  "test_id": 2,
  "score": 8
}
GET /results Получение результатов тестов.
Пример ответа:
{
  "data": [
    {
      "user_id": 1,
      "test_id": 2,
      "score": 8
    }
  ]
}
